# Нормативы качества воды

## Supluskohad (EU Bathing Water Directive 2006/7/EC)

Смысл показателей и отличия от бассейнов: [parametrid_ujula_avavee.md](parametrid_ujula_avavee.md).

| Параметр | Класс: Отлично | Класс: Хорошо | Класс: Допустимо | Запрет купания |
|----------|---------------|---------------|------------------|----------------|
| E. coli (внутр.) | ≤ 500 | ≤ 1000 | ≤ 900 | > 2500 |
| E. coli (морск.) | ≤ 250 | ≤ 500 | ≤ 500 | > 500 |
| Энтерококки (вн.) | ≤ 200 | ≤ 400 | ≤ 330 | > 1000 |
| Энтерококки (мор.) | ≤ 100 | ≤ 200 | ≤ 185 | > 500 |

Единицы: КОЕ/100 мл (PMÜ/100 ml)
