# Описание параметров качества воды

Этот документ описывает все параметры, измеряемые в пробах воды и используемые в модели.
Данные поступают от [Terviseamet](https://www.terviseamet.ee/) (Департамент здоровья Эстонии),
источник XML: [vtiav.sm.ee](https://vtiav.sm.ee).

---

## Микробиологические параметры

### `e_coli` — Кишечная палочка (E. coli)

| | |
|---|---|
| **Эстонский** | *Escherichia coli* |
| **Единица** | КОЕ/100 мл (колониеобразующих единиц на 100 мл) |
| **Нормы для купания** | ≤ 500 КОЕ/100 мл (внутренние воды, «отлично») |
| **Норма питьевой воды** | 0 КОЕ/100 мл |

Бактерия-индикатор фекального загрязнения. Сама по себе большинство штаммов безвредны, но
наличие E. coli сигнализирует о возможном присутствии патогенных микроорганизмов (сальмонеллы,
норовируса и т.д.). Для питьевой воды допуск нулевой: обнаружение одной колонии — нарушение.

**Почему важен в модели**: главный индикатор нарушений во всех доменах. При превышении нормы
`compliant = 0`.

---

### `enterococci` — Кишечные энтерококки

| | |
|---|---|
| **Эстонский** | *Enterokokid* |
| **Единица** | КОЕ/100 мл |
| **Норма для купания** | ≤ 200 КОЕ/100 мл (внутренние воды, «отлично») |
| **Норма питьевой воды** | 0 КОЕ/100 мл |

Грамположительные бактерии из кишечника людей и животных. Устойчивее к хлорированию, чем E. coli,
поэтому служат дополнительным маркером фекального загрязнения, особенно в морской воде. Директива
ЕС 2006/7/EC использует оба параметра (E. coli + энтерококки) для классификации купальных мест.

---

### `coliforms` — Колиформные бактерии (колиформы)

| | |
|---|---|
| **Эстонский** | *Kolibakterid / Coli-laadsed bakterid* |
| **Единица** | КОЕ/100 мл |
| **Норма питьевой воды** | 0 КОЕ/100 мл |

Широкая группа бактерий, включающая E. coli. Обнаружение колиформ при отсутствии E. coli может
указывать на загрязнение из почвы или на недостаточную дезинфекцию. Измеряется преимущественно
в сети водоснабжения (домен `veevark`).

---

## Химические параметры

### `nitrates` — Нитраты (NO₃⁻)

| | |
|---|---|
| **Эстонский** | *Nitraadid* |
| **Единица** | мг/л |
| **Норма питьевой воды** | ≤ 50 мг/л |

Нитраты поступают в воду из сельскохозяйственных удобрений и сточных вод. В желудке могут
превращаться в нитриты, вызывающие метгемоглобинемию («синий синдром» у младенцев). Повышенные
концентрации в грунтовых водах Эстонии связаны с интенсивным земледелием.

---

### `nitrites` — Нитриты (NO₂⁻)

| | |
|---|---|
| **Эстонский** | *Nitritid* |
| **Единица** | мг/л |
| **Норма питьевой воды** | ≤ 0.5 мг/л |

Промежуточный продукт окисления аммония (нитрификация) или восстановления нитратов. В сети
водоснабжения высокий уровень нитритов свидетельствует о неполной нитрификации или о разрастании
биоплёнки. Токсичнее нитратов при прямом воздействии.

---

### `ammonium` — Аммоний (NH₄⁺)

| | |
|---|---|
| **Эстонский** | *Ammoonium* |
| **Единица** | мг/л |
| **Норма питьевой воды** | ≤ 0.5 мг/л |

Появляется при разложении органических веществ, из удобрений, бытовых стоков. Косвенно указывает
на загрязнение органикой. При хлорировании воды аммоний реагирует с хлором, снижая эффективность
дезинфекции и образуя хлорамины.

---

### `fluoride` — Фторид (F⁻)

| | |
|---|---|
| **Эстонский** | *Fluoriid* |
| **Единица** | мг/л |
| **Норма питьевой воды** | ≤ 1.5 мг/л |

В небольших концентрациях (0.5–1.0 мг/л) фторид защищает зубную эмаль. Превышение нормы при
хроническом воздействии вызывает флюороз зубов и костей. В Эстонии повышенный фторид встречается
в некоторых скважинах Харьюского и Ляэнеского уездов.

---

### `manganese` — Марганец (Mn)

| | |
|---|---|
| **Эстонский** | *Mangaan* |
| **Единица** | мг/л (в XML иногда в мкг/л — конвертируется автоматически) |
| **Норма питьевой воды** | ≤ 0.05 мг/л |

Природный металл, встречающийся в подземных водах Эстонии. Окрашивает воду в бурый цвет, оставляет
осадок на сантехнике. При хроническом потреблении в высоких дозах негативно влияет на
нервную систему. Вместе с железом — одна из наиболее частых причин нарушений в системах
водоснабжения страны.

---

### `iron` — Железо (Fe)

| | |
|---|---|
| **Эстонский** | *Raud* |
| **Единица** | мг/л (в XML иногда в мкг/л — конвертируется автоматически) |
| **Норма питьевой воды** | ≤ 0.2 мг/л |

Широко распространён в подземных водах Эстонии (особенно в торфяных и песчаных горизонтах).
Придаёт воде металлический привкус и рыжий оттенок. Способствует образованию биоплёнок в
трубопроводах. Не является прямо токсичным в типичных концентрациях, но ухудшает органолептику
и технические свойства воды.

---

### `chlorides` — Хлориды (Cl⁻)

| | |
|---|---|
| **Эстонский** | *Kloriidid* |
| **Единица** | мг/л |
| **Норма питьевой воды** | ≤ 250 мг/л |

Естественный компонент воды. При концентрации выше нормы придаёт солоноватый вкус, ускоряет
коррозию металлических труб. Высокий уровень может указывать на засоление (приморские районы,
засоленные водоносные горизонты) или на промышленное загрязнение.

---

### `sulfates` — Сульфаты (SO₄²⁻)

| | |
|---|---|
| **Эстонский** | *Sulfaadid* |
| **Единица** | мг/л |
| **Норма питьевой воды** | ≤ 250 мг/л |

Содержатся в природных горных породах (гипс, ангидрит). При превышении нормы придают
горько-солоноватый вкус и могут вызывать расстройство кишечника. Высокие сульфаты характерны
для некоторых девонских и силурийских водоносных горизонтов Эстонии.

---

## Физико-химические параметры

### `ph` — Кислотность (pH)

| | |
|---|---|
| **Эстонский** | *pH* |
| **Единица** | безразмерная (шкала 0–14) |
| **Норма питьевой воды** | 6.5 – 9.5 |
| **Норма купания** | 6.0 – 9.0 |

Показатель кислотности / щёлочности воды. Значения pH вне допустимого диапазона разрушают
слизистые оболочки и раздражают кожу. Кислая вода (pH < 6.5) коррозионна и выщелачивает
свинец и медь из труб. Щелочная вода (pH > 9.5) снижает эффективность хлорирования.

В модели используется `ph_deviation` — отклонение от нормального диапазона [6.5, 9.5].

---

### `turbidity` — Мутность

| | |
|---|---|
| **Эстонский** | *Hägusus* |
| **Единица** | NTU (Nephelometric Turbidity Units) |
| **Норма питьевой воды** | ≤ 4 NTU |
| **Норма бассейна** | ≤ 0.5 NTU |

Мутность отражает количество взвешенных частиц (ила, органики, бактерий, коллоидов). Высокая
мутность снижает эффективность дезинфекции UV и хлором, так как частицы «прячут» патогены.
Является надёжным прокси-индикатором микробиологического загрязнения.

---

### `transparency` — Прозрачность (Secchi)

| | |
|---|---|
| **Эстонский** | *Läbipaistvus* |
| **Единица** | м (метры) |
| **Нормируется** | нет фиксированного числового норматива; оценивается ситуативно |

Измеряется диском Секки: глубина, на которой белый диск перестаёт быть виден. Применяется
исключительно для открытых купальных водоёмов (`supluskoha`). Низкая прозрачность указывает
на цветение водорослей, смыв почвы или органическое загрязнение. Директива ЕС 2006/7/EC
рекомендует визуальный контроль прозрачности воды на пляжах.

---

### `color` — Цветность

| | |
|---|---|
| **Эстонский** | *Värvus* |
| **Единица** | мг Pt/л (платиновая шкала, мг платины на литр) |
| **Норма питьевой воды** | ≤ 20 мг Pt/л |

Оценивает жёлто-коричневый оттенок воды, вызванный растворёнными гуминовыми и фульвовыми
кислотами (разложение торфа и листвы). В Эстонии высокая природная цветность воды характерна
для болотистых районов. Сама по себе цветность не токсична, но ухудшает органолептику и
является субстратом для образования хлороформа при хлорировании.

---

## Параметры бассейнов и СПА (домен `basseinid`)

Эти параметры измеряются только в пробах плавательных бассейнов, ванн и СПА. Нормы взяты
из постановлений Terviseamet для общественных бассейнов и отличаются от норм питьевой воды
и природных водоёмов (например, мутность в бассейне в 8× строже). Реализация в коде —
`NORMS_POOL` в `src/features.py`.

### `staphylococci` — Стафилококки (Staphylococcus aureus)

| | |
|---|---|
| **Эстонский** | *Stafülokokid* |
| **Единица** | КОЕ/100 мл |
| **Норма бассейна** | ≤ 20 КОЕ/100 мл |

*Staphylococcus aureus* — условно-патогенная бактерия, попадающая в воду с кожи, волос и
слизистых купальщиков. Высокие значения говорят о недостаточной гигиене посетителей,
перегрузке бассейна или сниженной эффективности дезинфекции. Особенно важный параметр для
ванн с тёплой водой и детских бассейнов, где сопротивляемость купальщиков ниже.

**Почему важен в модели**: один из двух ключевых бактериальных индикаторов домена `basseinid`
(вместе с `pseudomonas`). Используется производный признак `staphylococci_ratio`.

---

### `pseudomonas` — Синегнойная палочка (Pseudomonas aeruginosa)

| | |
|---|---|
| **Эстонский** | *Pseudomonas aeruginosa* |
| **Единица** | КОЕ/100 мл |
| **Норма бассейна** | 0 КОЕ/100 мл (не должна обнаруживаться) |

Патогенная бактерия, способная размножаться в системах циркуляции, фильтрах и биоплёнках
бассейнов даже при наличии хлора. Вызывает наружный отит («ухо пловца»), фолликулит и
кожные инфекции, особенно в гидромассажных ваннах (СПА). Её обнаружение требует немедленной
суперхлорировки и очистки системы.

Поскольку норма равна нулю, обычный `ratio` смысла не имеет — в модели используется бинарный
признак `pseudomonas_detected` (0/1) для домена `basseinid`.

---

### `free_chlorine` — Свободный хлор

| | |
|---|---|
| **Эстонский** | *Vaba kloor* |
| **Единица** | мг/л |
| **Норма бассейна** | 0.5 – 1.5 мг/л (диапазон) |

Активная форма хлора (HOCl / OCl⁻), обеспечивающая дезинфекцию воды. Это единственный
параметр с **диапазонной** нормой: слишком мало хлора → риск микробиологического загрязнения;
слишком много → раздражение кожи, глаз и дыхательных путей, образование побочных продуктов.

В модели рассчитывается `free_chlorine_deviation` — расстояние от разрешённого диапазона
[0.5, 1.5]: 0 означает норму, положительные значения — выход за пределы в любую сторону.

> **Phase 10 поправка.** Норма обновлена с прежнего диапазона `[0.2, 0.6]` на
> `[0.5, 1.5]` по результатам аудита `label-vs-norms` на 2194 пробах snapshot.
> Прежние значения давали 288 ложных срабатываний (`hidden_pass`) только по
> свободному хлору в бассейнах. Эмпирически compliant-пробы лежат в `[0.46, 1.6]`
> (p1–p99). Регуляторный источник: Sotsiaalministri 31.07.2019 määrus nr 49,
> Lisa 4. См. `docs/phase_10_findings.md`.

---

### `combined_chlorine` — Связанный хлор (хлорамины)

| | |
|---|---|
| **Эстонский** | *Seotud kloor* |
| **Единица** | мг/л |
| **Норма бассейна** | ≤ 0.5 мг/л (верхний предел) |

Хлорамины образуются при реакции свободного хлора с органическими загрязнителями —
мочевиной, потом, косметикой. Именно они отвечают за характерный «хлорный» запах бассейна
и раздражение глаз (а не сам хлор, как часто думают). Высокие значения говорят о
перегрузке органикой и требуют шокового хлорирования или частичной замены воды.

В модели: `combined_chlorine_ratio` = значение / 0.5; ratio > 1 указывает на нарушение.

> **Phase 10 поправка.** Норма обновлена с `≤ 0.4` на `≤ 0.5` по результатам
> аудита: compliant-пробы регулярно достигают 0.5 (p95), что не соответствует
> прежнему порогу. Источник: Sotsiaalministri 31.07.2019 määrus nr 49, Lisa 4.

---

### `oxidizability` — Окисляемость (перманганатная)

| | |
|---|---|
| **Эстонский** | *Oksüdeeritavus* |
| **Единица** | мг O₂/л |
| **Нормируется** | строгой численной нормы нет, оценивается ситуативно |

Интегральный показатель содержания легкоокисляемых органических веществ — косвенная оценка
органической нагрузки на воду. Повышенная окисляемость указывает на риск образования
побочных продуктов хлорирования (тригалометанов, хлороформа), а также на повышенный расход
дезинфектанта. Измеряется преимущественно в пробах бассейнов и СПА.

---

### `colonies_37c` — Общее микробное число при 37 °C

| | |
|---|---|
| **Эстонский** | *Kolooniad 37 °C juures* |
| **Единица** | КОЕ/мл |
| **Нормируется** | численной нормы нет, оценивается ситуативно |

Общее число колоний гетеротрофных бактерий, растущих при температуре тела (37 °C). Это
суммарный показатель бактериальной нагрузки, близкий к HPC (heterotrophic plate count) в
международных рекомендациях. Сам по себе не указывает на конкретный патоген, но резкий
рост значения говорит о сбое дезинфекции или формировании биоплёнки. Используется в
бассейнах как быстрый ориентировочный тест.

## Производные признаки модели

Из каждого измеренного параметра автоматически строятся производные признаки:

| Признак | Формула | Интерпретация |
|---|---|---|
| `<param>_ratio` | значение / норматив | > 1.0 означает нарушение |
| `ph_deviation` | max(0, pH_min − pH, pH − pH_max) | 0 = внутри нормы; > 0 = степень отклонения; границы зависят от домена (бассейн 6.5–8.5, vs 6.0–9.5) |
| `staphylococci_ratio` | staphylococci / 20 | только для `domain = basseinid` |
| `pseudomonas_detected` | 1 если > 0 КОЕ/100 мл, иначе 0 | бинарный флаг для `domain = basseinid` (норма = 0 не даёт обычный ratio) |
| `free_chlorine_deviation` | max(0, 0.5 − fc, fc − 1.5) | диапазонная норма: 0 = внутри [0.5; 1.5], > 0 = выход в любую сторону |
| `combined_chlorine_ratio` | combined_chlorine / 0.4 | только для `domain = basseinid` |
| `<param>_missing` | 1 если значение отсутствует | пропуск может означать «параметр не измерялся» |
